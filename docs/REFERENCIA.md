# Referencia

## Scripts Disponibles

| Comando | Descripción |
|---------|-------------|
| `yarn start` | Inicia el servidor Metro (bundler) |
| `yarn android` | Compila y ejecuta la app en Android |
| `yarn ios` | Compila y ejecuta la app en iOS |
| `yarn lint` | Ejecuta ESLint para verificar el código |

## Tecnologías Principales

- **React Native** 0.83
- **TypeScript**
- **React Navigation** (navegación)
- **Axios** (peticiones HTTP)
- **Ionicons** (iconos)

## Herramientas de Desarrollo

### Reactotron

El proyecto incluye **Reactotron** configurado para facilitar el debugging durante el desarrollo. Reactotron permite:

- Ver logs de la aplicación en tiempo real
- Monitorear las peticiones HTTP al backend (requests/responses)
- Inspeccionar el estado de la aplicación
- Trackear el rendimiento

**Instalación de Reactotron:**

1. Descargar Reactotron desde: https://github.com/infinitered/reactotron/releases
2. Instalar la aplicación de escritorio
3. Ejecutar Reactotron antes de iniciar la app
4. La conexión se establece automáticamente al iniciar la app

> La configuración se encuentra en `ReactotronConfig.js` en la raíz del proyecto.
